🎨 제2탄: [UI/UX 설계서] 프로젝트 Clib (클립)
1. 디자인 아이덴티티

톤 앤 매너: 생산성 도구다운 깔끔함 + 게임 같은 경쾌함

컬러 시스템:

Main: Deep Indigo (신뢰감)

Point: Neon Green (완료/긍정), Soft Coral (보류/부정)

테마: 다크 모드(Primary), 라이트 모드(Secondary) 지원 (System setting 연동)

2. 화면별 상세 설계

[화면 0: 공유 확장 팝업 (Share Extension)]

구성: 상단 썸네일(소), 중앙 제목, 하단 라벨 선택 칩(Horizontal Scroll)

UX: 공유 버튼 누르자마자 0.5초 내에 팝업 노출. 라벨 선택은 선택 사항이며 [저장] 누르면 즉시 팝업 닫힘.

[화면 1: 홈 탭 (Home - Swiping)]

컴포넌트: CardStack

카드 구성: 전체 화면의 70% 차지하는 대형 카드. 배경은 아티클 썸네일, 하단에 제목과 플랫폼 뱃지 오버레이.

인터랙션:

오른쪽 스와이프 시: 카드 테두리에 Neon Green 하이라이트 발생하며 사라짐.

왼쪽 스와이프 시: 다음 카드로 넘어가고 현재 카드는 스택의 가장 아래로 이동.

[화면 2: 보관함 탭 (Library - Grid)]

구성: 2열 그리드 레이아웃.

카드 구성: 라벨명 + 아이콘 + 하단 원형 프로그레스 바.

상세 페이지: 특정 라벨 클릭 시 해당 라벨의 아티클들이 리스트 형태로 나열됨. [안 읽음 / 다 읽음] 필터 탭 존재.

[화면 3: 설정 탭 (Settings)]

푸시 관리: 라벨 리스트 옆에 [알림 설정] 버튼. 클릭 시 요일(월~일 다중선택)과 시간을 고르는 Time Picker 노출.

테마 설정: 시스템 설정 / 라이트 / 다크 모드 선택 라디오 버튼.

3. 주요 인터랙션 가이드

햅틱 피드백: 스와이프 완료 시, 저장 완료 시 가벼운 진동(Haptic Feedback) 제공으로 '지식을 처리했다'는 손맛 강조.

애니메이션: 카드가 넘어갈 때 부드러운 가속도(Curve) 적용.
